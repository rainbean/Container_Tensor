# Purpose

Baked machine learning framework stack, CUDA backend

### Python packages

- Ubuntu 18.04
- CUDA 10.2 (require nivida 440+ driver)
- Python 3.6.9
- PyTorch 1.8.1
- TorchVision 0.9.1
- Tensorboard 2.4.0
- Numpy 1.19.5
- Pandas 1.1.5
- Pillow 8.1.2
- Scikit-image 0.17.2
- OpenCV 4.5.1
- Jupyter-Lab 3.0.12

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```

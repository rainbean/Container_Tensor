# Purpose

Baked machine learning framework stack, CUDA backend

### Python packages

- Ubuntu 18.04
- CUDA 10.1 (require nivida 418+ driver)
- Python 3.6.8
- PyTorch 1.3.0
- Numpy 1.17.0
- Matplotlib 3.1.1
- Scipy 1.3.1
- Pandas 0.25.0
- Scikit-learn 0.21.3
- Scikit-image 0.15.0
- OpenCV 4.1.0

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```

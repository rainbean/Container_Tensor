# Purpose

Baked machine learning framework stack, CUDA backend

### Packages

- Ubuntu 20.04
- CUDA 11.1 (require nivida 440+ driver)
- Python 3.8.10
- PyTorch 1.8.2 LTS
- TorchVision 0.9.2
- Tensorboard 2.9.1
- Numba 0.55.2
- Numpy 1.22.4
- Pandas 1.4.3
- Pillow 9.2.0
- Scikit-image 0.19.3
- OpenCV 4.6.0
- Jupyter-Lab 3.4.3

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```

# Purpose

Baked machine learning framework stack, CUDA backend

### Python packages

- Ubuntu 18.04
- CUDA 10.2 (require nivida 440+ driver)
- Python 3.6.8
- PyTorch 1.5.1
- Numpy 1.18.5
- Matplotlib 3.1.3
- Scipy 1.5.0
- Pandas 1.0.5
- Scikit-learn 0.23.1
- Scikit-image 0.16.2
- OpenCV 4.2.0

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```

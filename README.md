# Purpose

Baked machine learning framework stack, CUDA backend

### Python packages

- Ubuntu 18.04
- CUDA 10 (require nivida 410+ driver)
- Python 3.6.7
- TensorFlow 1.13.1
- PyTorch 1.0.1
- JupyterLab
- Numpy 1.16.1
- Matplotlib
- Scipy 1.2.1
- Pandas 0.24.1
- Scikit-learn 0.20.2
- Scikit-image 0.14.2
- OpenCV 4.0
- Polyline

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```

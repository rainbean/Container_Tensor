# Purpose

Baked machine learning framework stack, CUDA backend

### Python packages

- Ubuntu 20.04
- CUDA 11.0 (require nivida 450+ driver)
- Python 3.8
- PyTorch 1.7.1
- TorchVision 0.8.2
- Tensorboard 2.4.0
- Numpy 1.18.5
- Pandas 1.1.0
- Pillow 8.0.1
- Scikit-image 0.18.1
- OpenCV 4.4.0

### Usage

Docker pull command
```
$ docker pull rainbean/tensor
```
